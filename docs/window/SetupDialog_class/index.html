    showSetupDialog #

    bool showSetupDialog(MeasurementSetup &setup)

    被调用时会显示设置窗口

    参数：

    • setup：MeasurementSetup类的引用

    返回值：用户点击确定(ok)按钮关闭窗口返回true，其它情况返回false
